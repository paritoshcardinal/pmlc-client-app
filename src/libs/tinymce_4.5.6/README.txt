PML Tinymce upgradtion guide

1) Dowload the latest tinymce
2) Make the below changes in tinymce.min.js to fix PML-3811 ticket
f.contentStyles.push("body {min-height: 150px}") replace this line by below
f.contentStyles.push("body {min-height: 15px}")/*PML-Customization*/  # just make min-height 15px and add comment /*PML-Customization*/

3)Add below file related to equation editor for Math tool at root level
config.js
equation_editor.html

4) Add Math tool plugin folder named "equationeditor"(that have only one file i.e. "plugin.min.js") under plugins folder
5) Update the path "equation_editor.html" in file "/tinymce/plugins/equationeditor/plugin.min.js"
6) Add below css in "/tinymce/skins/lightgray/skin.min.css" for PML equation editor UI theme or take css code from old skin.min.css
